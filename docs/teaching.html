Teaching Experience
  • I enjoy teaching and have served as a TA for the following:
    • Introduction to Computer Security [2023, 2024, 2025]. I have provided project support for the following topics: One Time Pad reuse, Length Extension attacks, subverting HTTPS usage through network packet analysis, Password Cracking, TOCTOU exploits, Buffer Overflows, Certification Measurement, and Fuzzing using AFL.
      I have also led discussion sections covering cryptography (Symmetric Encryption, Steam Ciphers, Block Ciphers, and Cryptographic Hash Functions).
    • Engineering for Ethics, Privacy, and Fairness in Computer Systems [2024]. I guest-lectured and otherwise provided support for following topics: LLMs and their safety/guardrails, k-anonymity, and differential privacy.
    • Inclusive Technology [2022, 2025]: Designing for Underserved and Marginalized Communities. I guided and mentored three groups of students' research projects through weekly check-ins.
    • Introduction to Computer Systems [2022]. I helped students create higher-level programming constructs (e.g., binary search) at the bit level.
